# ROBLOX_COMBAT_TEST

## Azul installation

1. Install Node.js from [nodejs.org](https://nodejs.org/).
2. install Azul globally:

   ```sh
   npm install azul-sync -g
   ```

3. Install the Azul Companion Plugin for Roblox Studio from the [Roblox Create store](https://create.roblox.com/store/asset/79510309341601/Azul-Companion-Plugin).
4. With the terminal in the project folder, run:

   ```sh
   azul
   ```

5. In Roblox Studio, click on the **Connect** button in the Azul plugin.
6. Start coding!
